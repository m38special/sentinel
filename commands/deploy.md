# /deploy - Deploy to Production

Deploys LiQUiD SOUND services to production.

## Usage
/deploy [service]

## Services
- sentinel — SENTINEL token scanner
- liquidswap — Liquid.Swap DEX
- uai — Universal Agent Interface

## Steps
1. Check git status
2. Push to phase5 branch
3. Verify deployment
4. Post to Slack

## Examples
/deploy
/deploy sentinel
/deploy liquidswap
