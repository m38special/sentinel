# Liquid.Swap

Solana token swap interface powered by Jupiter Aggregator.

## Features
- Connect Solana wallets (Phantom, Solflare, Backpack)
- Instant token swaps via Jupiter API
- Real-time quotes and price impact
- Customizable slippage tolerance

## Development
```bash
# Open index.html in browser
```

## Deployment
- Netlify: Drag & drop folder to https://app.netlify.com/drop
- Railway: Connect GitHub repo

## Tech Stack
- Vanilla JS (no build required for MVP)
- Jupiter Aggregator API
- Solana Wallet Adapter
