# Liquid.Swap - Solana Token Swap Platform

## Overview
A decentralized exchange interface for swapping SPL tokens on Solana blockchain, powered by Jupiter Aggregator.

## Tech Stack
- **Frontend:** React + Vite
- **Wallet:** @solana/wallet-adapter
- **Swap API:** Jupiter Aggregator V6
- **Styling:** Tailwind CSS
- **Deployment:** Netlify/Vercel

## Core Features

### 1. Token Swap
- Connect Solana wallets (Phantom, Solflare, Backpack)
- Select input/output tokens
- Real-time price quotes from Jupiter
- Slippage tolerance configuration
- Execute swap transactions

### 2. Token List
- Popular tokens (SOL, USDC, USDT, BONK, WIF, etc.)
- Search functionality
- Token balance display
- Favorite tokens

### 3. Transaction History
- Recent swaps
- Status tracking (pending/success/failed)

### 4. Analytics
- Token prices
- Volume indicators
- Price impact warnings
