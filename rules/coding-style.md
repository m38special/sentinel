# Coding Style Rules

## Python
- Follow PEP 8
- Use type hints
- Max line length: 100
- Docstrings: Google style

## JavaScript
- Use ES6+
- Prefer const/let over var
- Semantic naming

## General
- Small, focused functions (< 50 lines)
- No magic numbers
- Meaningful variable names
- Comments: why, not what

## File Organization
- One class per file
- Group related modules
- Clear directory structure
